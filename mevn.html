MEVN Chat- Web Application (Individual Project):

Home MEVN Chat

Project Description

A simple chat application using MEVN (Mongo, Express, Vue, Node) and Socket.io. This chat application is deployed with heroku and also uses axios. The main page shows the list of rooms and option to join. If the user clicks on join it asks them for their nickname and finally it takes the user to the chat room with other users.

Home Page

The home page consists of list of the chatrooms. It also provides user to create a new chatroom. The user can click on the join room button and join the chat room and starting chat

Chat Page

The admin dashboard consists of all the CRUD functionality. The chat page shows all the messages and the users in that particular chat. The user can write the message and send. The user can logout and can join many other rooms.
