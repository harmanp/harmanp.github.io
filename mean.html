Gamers Lobby- Web Application (Individual Project):

Deployed on Heroku       On Github  

Home GamerLobby Project Details

Project Description

A web application developed using MEAN Stack for players to join other online available players to play games. This application features a list of available games that the players can join if they are interested. There is an admin interface for creating and monitoring players and games.

Highlights of the Project

  • Constructed REST APIs for the backend in ExpressJs with routes separating the business logic and Schema definition in models.
  • Created NoSQL database on MongoDB using tools like MongoDB Compass.
  • Authentication is done through JSON Web Token which is encoded with user credentials.
  • User password encryption done through bcrypt technology before storing them to MongoDb database.
  • Designed the Front-end AngularJs 8.2 app using Angular Material design.
  • Followed single responsibility components on the front-end app like add-game, add-player components.
  • Interacts with backend APIs using a shared interface in Typescript for all the components.
  • Guarded routes through CanActivate interface in Angular for restrictive access to the pages only after successful authentication.
  • Designed, Developed, Tested, debugged and deployed in one week successfully with all requirements provided by the teacher.

Main Page

The home page consists of lists of players avaible and guest can choose any player to join the game. The join button renders a new page for selecting the available players from the mongodb database. If a player successfully joins the game the status is updated in the database to unavailable. If clicked on admin tab the page will also provides the oppurtunity for admin to login.

Admin dashboard

The admin dashboard consists of all the CRUD functionality. The admin can view players and games by clicking on the respective tabs. The page also provides admin an oppurtunity to add a new players or games. The admin dashboard also provides the oppurtunity for admin to edit or delete an existing players or games.
