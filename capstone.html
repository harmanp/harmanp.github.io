Home Capstone Project

Project Description

An android application for the George brown college students to organize events and contact each other. This is a group project which I was involved along with three of my classmates. Developed on Android Studio with features such as creating and sharing posts, helping students to organize events and communicate. This application will welcome you with the main screen where u are given the option either to login or register. This application also provides the admin features where admin can control the users and events.

Login Page

The login page consists three options first is to create an account, if you already have an account then login with the credentials. If you have forgotten the password then you can reset your password. The email will be verified if its georgebrown email and password should also match the one with database.

Home

This page is the home page which you will recieve once you login. The page shows all the events which are available to join. If clicked on particular event it will show you the details of the event. The top bar menu shows you a drawer where you will recieve more options like my profile, logout etc. This page also provides search and sort options for the list. The bottom menu have options like home,myevents and messages.
